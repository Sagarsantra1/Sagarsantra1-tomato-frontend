## Overview

Tomato is a Zomato clone, offering a familiar and intuitive user experience for exploring and ordering from local restaurants. It's designed to simplify the food ordering process, provide comprehensive restaurant information, and enhance the overall dining experience.



## Features

- **Explore Cities**: Browse and select from a list of available cities.
- **Browse Restaurants**: View restaurant cards with essential details.
- **Sort and Filter**: Customize your restaurant listings by sorting and applying filters.
- **Restaurant Details**: Access detailed restaurant information, including menus and food photos.
- **Search Functionality**: Search for restaurants and cuisines using the search input.
- **City Selection**: Easily change your current city for exploring different areas.

## Technologies

Tomato is built using a stack of modern web technologies:

- **React.js**: A powerful library for building dynamic user interfaces.
- **React Router**: Handling routing and navigation within the app.
- **React Query**: Efficient data fetching and state management.
- **React Icons**: Customizable icons to enhance the UI.
- **Tailwind CSS**: A utility-first CSS framework for responsive and stylish designs.

## Getting Started

To begin using Tomato and explore local restaurants, follow these steps:

### Backend_Repository
You can find the backend code for this project in the

[Link to Backend Repository](https://github.com/Sagarsantra1/tomato-backend)

### Prerequisites

Before you start, ensure you have the following software installed:

- Node.js (v14 or higher)
- npm (Node Package Manager)

### Installation

1. Clone the Tomato repository:

   ```bash
   git clone https://github.com/Sagarsantra1/tomato-frontend.git
   cd tomato/
   ```

2. Install project dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

## Usage

Tomato provides a user-friendly experience for exploring and ordering food:

### Exploring Cities

- On the homepage, you'll find a list of available cities.
- Click on a city to discover restaurants in that area.

### Browsing Restaurants

- Browse through restaurant cards to see essential details.
- Sort restaurants based on your preferences.
- Apply filters to find the perfect dining option.

### Viewing Restaurant Details

- Click on a restaurant card to access detailed information.
- Explore menus, view food photos, and read reviews.

### Search and Navigation

- Use the search input in the header to find specific restaurants or cuisines.
- Navigate between different pages and explore offerings easily.

### Changing City

- Change your current city using the options available in the header.
- Explore a variety of restaurants in different locations.
